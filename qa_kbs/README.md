# qa_kbs — 通用備詢知識庫

政府標案評選「備詢／統問統答」的**跨案共用**知識庫。從各標案 `備詢準備-*` 檔抽出**與特定標案無關的通用內容**（法規事實、資安定調、通用話術與方法論），未來新標案可直接取用。

## 通用 vs 專案分工
- **這裡（qa_kbs/）**：法規時限、採購法條、ISO 控制項、資安定調、無障礙規範、資料工程方法論、通用話術。
- **各標案資料夾（如 `yilan-bids/備詢準備-*`）**：只留該案專屬——具體金額、時程節點、案號、客戶佐證（法雅客／TAP 等）、特有業務規則（監護權、候補序位…），並在需要通用依據處**引用本庫**。

## 使用約定
- 檔內以 `[[檔名]]` 互相連結；答題時把「本案／機關」換成該標案脈絡。
- 法規數字（時限、金額、比率、ISO 控制項編號）為查核後事實，**引用勿改**；有更新時在本庫一處修訂即可。
- ⚠️ 「業界標準做法」不等於「本案已做」——每案上場前確認哪些已做、哪些要為本案補齊。
